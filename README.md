# Santinitiative

Tracker di iniziativa per D&D con input vocale.

Detta coppie come:

- `Roy 19`
- `Taurion24`
- `Boss32`
- `Giucan 7`

L'app ordina automaticamente dal numero piu alto al piu basso.

## Pubblicazione

Questa app non richiede backend. Pubblica tutti i file di questa cartella con GitHub Pages, Cloudflare Pages, Netlify o Vercel.
